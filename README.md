# Hedegreen Research Transparency

This is the separate declarations repo for Hedegreen Research.

It exists to hold the public transparency ledger without exposing the wider private workspace, the full site source, or unrelated project history.

## What belongs here

- `ledger.json`
- redacted contracts when they exist
- declaration attachments when they are relevant
- small notes that clarify the declaration model

## What does not belong here

- the wider Hedegreen Research workspace
- private mail or contact logs
- unpublished article drafts
- LifeOS state
- site runtime code

## Current rule

This repo is now the public source-of-record for the Hedegreen Research declaration ledger.

The live V2 site may still serve its own public `ledger.json` copy, but that copy should stay in sync with the file here.

The intended model is:

1. this repo holds the public declaration history
2. the V2 site links to the GitHub source and history here
3. redacted contracts or declaration attachments can be added here without exposing the wider workspace
